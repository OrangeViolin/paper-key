# 纸上键 · 安装指南（内测版）

> 给你的键盘一百种灵魂。敲出真实的机械音、禅意的敲击声或复古的打字机声。

---

## 1. 安装

1. 双击 `纸上键-v0.1.0.dmg`
2. 把 **纸上键.app** 拖到 **Applications** 文件夹

## 2. 第一次打开（重要）

因为这是内测版，没有走 Apple 正式公证，macOS 会拦一下。**只要一次**：

- 打开「访达」→ 应用程序 → 找到 **纸上键**
- **右键 → 打开** （不是双击！）
- 弹窗问「仍要打开吗」→ 点 **打开**

之后双击就能直接用了。

## 3. 授权「输入监控」

app 里会有个三步引导带你走完。核心是一步：

**系统设置 → 隐私与安全性 → 输入监控 → 开启「纸上键」**

授权后，在任何 app 里敲键盘都会有声。

## 4. 万一不出声了

打开 app，点右上角状态条（会显示 🟡 权限失效），按面板提示操作。

最常见的情况：**重装/升级 app 后监听失效**。操作：
1. 点「打开输入监控」
2. 列表里把「纸上键」**删掉**（选中按 `-`）
3. 再把 `/Applications/纸上键.app` 拖回列表
4. 回到 app 点「重启 app」

## 5. 卸载

从 Applications 里拖到废纸篓即可。
